# Der Ball in Hall

A small browser game starter built for this new workspace.

## Play locally

1. Open [index.html](index.html) in a browser, or
2. Run a local server from this folder and visit http://localhost:8000

## Controls

- Arrow keys or WASD to move
- Reach the green exit area
- Avoid the red barriers

## Files

- [index.html](index.html) for the page structure
- [style.css](style.css) for styling
- [script.js](script.js) for game logic
